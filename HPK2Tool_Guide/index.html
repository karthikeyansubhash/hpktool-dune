HP Workpath Solution Utility

HPK2Tool вҖ” Developer Guide

v2.0-beta10.0 Build 20260422 HPK v2.5 Platform 31.8

The HP Workpath Solution Utility (HPK2Tool) is a developer tool for packaging, deploying, and managing HP Workpath solution bundles (.bdl / .hpk) on HP Workpath-capable MFPs and printers. It takes an Android Package (.apk) and metadata options, and produces a signed, deployable solution bundle ready to install over the network.

HPK2Tool ships as both a Command-Line Interface (CLI) and a Graphical User Interface (GUI), supporting Windows, Linux, and macOS.

Prerequisites

The Solution Packaging Utility CLI tool is supported in the Windows, Linux, and Mac environments. The system requirements for running the tool are:

  • An x64/x86 machine with at least 8 GB of RAM and 10 GB of free disk space.
  • .NET Desktop Runtime 8 must be installed.

Windows Requirements

  • Windows 10 or newer.
  • Windows Subsystem for Linux (WSL) 1 using the Ubuntu 20.04 Linux distribution as your default WSL container.
рҹ“Ң
Note вҖ” WSL 1 Required WSL 1 is required due to the OXPd2 solution bundle format utilizing squashfs and dm-verity in the build process. WSL 2 is not supported.

Installing WSL 1 with Ubuntu 20.04

  1. Open PowerShell as Administrator and run:
wsl --install -d Ubuntu-20.04
wsl --set-version Ubuntu-20.04 1
wsl --set-default Ubuntu-20.04
  1. Verify: wsl --list --verbose вҖ” the VERSION column must show 1.

Installing .NET Desktop Runtime 8 on Windows

  1. Visit https://dotnet.microsoft.com/en-us/download/dotnet/8.0.
  2. Download the .NET Desktop Runtime 8.x.x вҖ” Windows x64 Installer.
  3. Run the installer and follow the prompts.
  4. Verify in a new terminal: dotnet --version вҶ’ should print 8.x.x.

Linux Requirements

  • Debian 9+, Ubuntu 16.04+, or Fedora 30+

Installing .NET Runtime 8 on Ubuntu / Debian

sudo apt-get update
sudo apt-get install -y dotnet-runtime-8.0

Or download the binary tarball directly from dotnet.microsoft.com and extract it to a directory on your PATH.

Installing .NET Runtime 8 on Fedora

sudo dnf install dotnet-runtime-8.0

Verify

dotnet --version   # should print 8.x.x

macOS Requirements

  • macOS High Sierra 10.13 or newer.

Installing .NET Runtime 8 on macOS

  1. Visit https://dotnet.microsoft.com/en-us/download/dotnet/8.0.
  2. Download the .NET Runtime 8.x.x вҖ” macOS x64 (Intel) or Arm64 (Apple Silicon) installer.
  3. Run the .pkg installer and follow the prompts.
  4. Verify: dotnet --version

Architecture Overview

HPK2Tool is a Kotlin/JVM multi-module application. The CLI and GUI share a Common Library for all services, data models, and utilities. Both tools produce the same output вҖ” a signed solution bundle ready to deploy to an HP Workpath device.
