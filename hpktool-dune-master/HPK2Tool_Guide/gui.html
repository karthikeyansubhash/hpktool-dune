GUI Guide

HP Workpath Solution Utility — Graphical Interface Reference

v2.0-beta10.0 JavaFX · 650×600 window

Overview

The HP Workpath Solution Utility GUI provides the same capabilities as the CLI but through a graphical interface. It is organized as a single main window with a persistent menu bar, and navigates between functional screens using the Tools menu.

The GUI covers four main workflows:

  • Build — Create a new solution bundle from an APK (Solution Builder).
  • Deploy — Install or uninstall a bundle on a device (Install / Uninstall).
  • Manage — Browse, inspect and remove solutions on a device (Management).
  • Configure — Read or update solution config and attestation data (Configuration, Attestation).

Launching the GUI

"HPK2Tool.exe"
./HPK2ToolGui-linux.sh

The application opens at 650 × 600 px and defaults to the Solution Builder screen.

End-to-End Workflow

The following steps walk through a complete workflow: building a solution bundle in the GUI and deploying it to a device.

Workflow — Build and Deploy a Solution

  1. Launch the HP Workpath Solution Utility and confirm it opens on the Solution Builder screen (or choose File › New).
  2. Select APK: In the Solution tab, click Select File next to APK File and choose your .apk.
  3. Fill Solution Metadata: Enter Solution ID (click Generate), Solution Name, Description, Vendor, Version, Version Number, Date, and select a Platform Version from the dropdown.
  4. Add an Application: Click Add Application, fill in the Application Dialog, and click Add.
  5. Enable Components (optional): Check Scan Agent, Authentication, etc. in the Component Selector to add those capabilities to the bundle. Fill in any additional fields that appear in their respective tabs.
  6. Build: Click Create Bundle. The progress indicator spins during the build. On success, a status message shows the output path. Use the dropdown arrow to also generate an .hpk file.
  7. Install: Go to Tools › Install / Uninstall. Enter the device IP and password, select the generated .hpk2 file, and click Install. (Also Install .bdl is possible.)
  8. Verify: Go to Tools › Management. Enter the device IP and password, click Enumerate, and confirm your solution appears in the list. Click Detail to inspect it.
💡
Tip — Re-open an Existing Bundle Use File › Open to load an existing .bdl, .hpk2 back into the Solution Builder for editing and re-building. Use File › Open HPK for legacy .hpk bundles.
